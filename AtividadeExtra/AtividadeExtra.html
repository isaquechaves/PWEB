Java

Java é uma linguagem multiplataforma, orientada a objetos e centrada em rede que pode ser usada como uma plataforma em si. É uma linguagem de programação rápida, segura e confiável para codificar tudo, desde aplicações móveis e software empresarial até aplicações de big data e tecnologias do servidor.

Spring Boot

Spring Boot é um framework para desenvolvimento de aplicações em Java que se tornou cada vez mais popular nos últimos anos. Ele foi criado para tornar o desenvolvimento mais rápido e fácil, fornecendo uma configuração simplificada e predefinida para muitos dos componentes do ecossistema Spring.

Microsserviços

Os microsserviços promovem uma organização de equipes pequenas e independentes que são proprietárias de seus serviços. As equipes atuam dentro de um contexto pequeno e claramente compreendido e têm autonomia para trabalhar de forma mais independente e rápida. O resultado é a aceleração dos ciclos de desenvolvimento.

DevOps

DevOps é um conjunto de práticas de integração entre equipes de desenvolvimento de software, operações e segurança, que tem como objetivo gerar valor ao cliente. A principal prática é a adoção de processos automatizados para produção rápida e segura de aplicativos e serviços.
